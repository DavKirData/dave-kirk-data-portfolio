# Supply Chain KPI Dashboard (Power BI)

## Overview
This project demonstrates a Power BI dashboard designed to track key warehouse performance metrics including throughput, picking accuracy, labor productivity, and on-time shipping.

## Tools Used
- Power BI
- Power Query
- DAX
- SQL

## Key Insights
- Identified bottlenecks in picking workflow
- Improved labor allocation through hourly productivity tracking
- Highlighted variance between shifts and order types

## Files
- /data – sample datasets
- /powerbi – dashboard screenshots or PBIX file
- /sql – queries used for data preparation

## Skills Demonstrated
Data modeling, DAX, KPI design, operational analytics, visualization
